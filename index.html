Legend

  1. Enter a valid username in GitHub
  2. Press the search button (or ↵)
  3. Hover over the repository's name, and you will see it's description
  4. Repositories with a ⭐ indicate a repository that has been starred
  5. Repositories with a 🍴 indicate a repository that has been forked from
  6. Repositories with a 👁 indicate a repository that has watchers
  7. Hover over any emoji (⭐, 🍴, 👁) to see the amount the repository has of that item
